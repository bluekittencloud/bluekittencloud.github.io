Windows游戏专版(SSTap)教程

  • Windows游戏专版(sstap)


    1. 点击下载Windows游戏专版(sstap)；

    2. 下载然后安装，安装完成后，打开 SSTap， 点击 「设置」-「SSR订阅」-「SSR订阅管理…」；


    3. 在鸡场的用户中心里，复制节点订阅链接；

    4. 粘贴节点订阅链接至「URL」中，点击「添加」，然后关闭返回到主界面；


    5. 选择一个合适的节点；


    6. 模式」一般选择「仅网页浏览器（跳过中国站点）」，适合日常上网，如不能使用，则切换「全局」即可；


    7. 最后点击「连接」即可科学上网。
