windows（SSR）使用教程

  • Windows（SSR）


    1. 点击下载Windows（SSR）；

    2. 下载解压后打开ShadowsocksR-dotnet4.0.exe，打不开就用2.0，2.0打不开请下载安装 net.framework,点击安装，还不能安装麻烦升级到win7；

    3. 找到系统托盘菜单中纸飞机图标右键调出菜单，将代理模式设置为直连，负载均衡勾掉；

    4. 在鸡场的用户中心里，复制节点订阅链接；

    5. 打开SSR服务器订阅链接设置；

    6. 点击add添加一个订阅，将复制的链接填入右侧框内点击确定；

    7. 桌面纸飞机图标右键调出菜单，点击更新SSR服务器订阅(不通过代理)；

    8. 服务器->找到对应本站的节点组->选择一个节点单击；

    9. 找到系统托盘菜单中纸飞机图标右键调出菜单，将代理模式设置为PAC(推荐)/全局；

    10. 不出意外，可以实现科学上网了。
